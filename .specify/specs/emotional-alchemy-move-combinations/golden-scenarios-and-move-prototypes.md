# Golden Scenarios and Move Card Prototypes v0.1

## Purpose

This artifact restarts the interrupted design move without collapsing the axes.

It is not one scenario deck. It is a test bench:

- 12 golden player scenarios.
- Separate superpower and domain tables.
- 20 move card prototypes generated from clean axis intersections.

The point is to test:

- Same scenario + different superpower = different expression.
- Same vector + different domain = different output.
- Same card + different blocker = different move.

## Rules For This Artifact

- A scenario is not a superpower.
- A domain is not a superpower.
- A card is not the whole move.
- A blocker changes the move even when the card stays the same.
- Every prototype must name its emotional vector, superpower, domain, card/context, blocker, mechanic, and completion condition.

## Superpower Table

| Superpower | What It Changes | Clean Move Bias | Shadow To Watch |
|---|---|---|---|
| Connector | Relationship, trust, bridges, belonging, access | Builds consent-aware connection and routes support through real trust. | Over-connecting, absorbing everyone's emotions, confusing access with intimacy. |
| Strategist | Sequence, leverage, constraints, pattern | Finds the smallest sequence that makes action clearer. | Over-control, analysis paralysis, treating people like pieces. |
| Disruptor | Silence, compliance, false inevitability | Interrupts the pattern that is protecting harm or stuckness. | Escalation for identity, spectacle, damaging trust the mission still needs. |
| Escape Artist | Options, exits, refusal, alternate routes | Creates a safe way to pause, leave, refuse, or move laterally. | Avoidance, slipping accountability, treating every agreement as captivity. |
| Catalyst / Coach | Capacity, practice, courage, activation | Names the smallest practice rep that brings power online. | Over-coaching, dragging people, making others' growth carry self-worth. |
| Alchemist | Consequence, residue, failure, stuckness | Converts emotional residue into ritual, repair, resource, or renewed motion. | Romanticizing suffering, overprocessing, transforming before witness. |
| Storyteller | Meaning, memory, morale, public truth | Shapes a truthful arc that helps people understand and act. | Spin, simplification, performance, story replacing material change. |

## Domain Table

| Domain | What Output Looks Like | Domain Question |
|---|---|---|

| Direct Action | Intervention, refusal, interruption, boundary, concrete move | What line or move is being avoided? |
| Skillful Organizing | Sequence, role, handoff, structure, agreement, decision path | What structure would let the work continue? |

## Test Views

### Same Scenario + Different Superpower

| Scenario | Compare | What Should Change |
|---|---|---|
| GS01 | MP01 Connector vs MP02 Strategist vs MP03 Catalyst/Coach | Connector routes through trust and consent; Strategist bounds the ask; Catalyst turns the ask into a practice rep. |
| GS02 | MP04 Storyteller vs MP05 Disruptor | Storyteller protects truth and consent; Disruptor interrupts silence. |
| GS03 | MP06 Strategist vs MP07 Connector | Strategist creates sequence; Connector creates trustable handoff. |
| GS05 | MP10 Alchemist vs MP11 Storyteller | Alchemist metabolizes failure into renewed motion; Storyteller makes the meaning actionable. |

### Same Vector + Different Domain

| Vector | Compare | What Should Change |
|---|---|---|
| anger -> triumph | MP12 Direct Action vs MP13 Skillful Organizing | Direct Action interrupts the pattern; Skillful Organizing prevents repetition. |
| fear -> peace / clean fear | MP16 Skillful Organizing vs MP17 Gather Resources | Organizing clarifies depletion and decision; Gathering Resources activates support. |
| sadness -> joy / support | MP14 Direct Action vs MP15 Gather Resources | Direct Action closes distance one inch; Gathering Resources asks for shared support. |

### Same Card + Different Blocker

| Card / Context | Compare | What Should Change |
|---|---|---|
| Show Up + Gather Resources | MP01 vs MP02 vs MP03 | Consent blocker creates bridge mechanics; vagueness blocker creates constraint mechanics; readiness blocker creates practice-rep mechanics. |
| Show Up + Direct Action | MP08 vs MP10 vs MP20 | Exit blocker creates route mechanics; failure blocker creates compost mechanics; repair blocker creates accountability mechanics. |
| Show Up + Skillful Organizing | MP06 vs MP13 vs MP19 | Confusion blocker creates sequence; recurrence blocker creates prevention structure; overfunctioning blocker creates handoff. |

## Immediate Design Questions

- Should Catalyst become the user-facing label while Coach remains the code key?
- Should every prototype attach to an existing generated Allyship Deck card ID, or should Emotional Alchemy Show Up moves become a separate card layer that can cite deck cards as context?
- Are natural superpower-domain fits enough for the first playtest, or do we need a small number of deliberate off-fit examples to prove the axes can recombine?
