# Copy audit — Lenses / Observatory (PMA B2.3)

**Purpose:** Ensure Lenses and Observatory copy never reads as a productivity planner.  
**Authority:** [SIX_GAME_MASTER_REVIEW.md](./SIX_GAME_MASTER_REVIEW.md), [productivity-modality-alignment/SIX_GAME_MASTER_REVIEW.md](../productivity-modality-alignment/SIX_GAME_MASTER_REVIEW.md).

## Forbidden (reject or rewrite)

| Phrase / pattern | Why |
|------------------|-----|
| inbox zero | Shame-spiral; GTD cosplay |
| you're behind / catch up | Black-hat pressure |
| streak broken / don't break the chain | Punitive unless opt-in celebratory (POF SPC only) |
| productivity / planner / task manager | Portland trust; LWX veto |
| Todoist / Notion / ClickUp / Mindwtr / Tandem (player UI) | Competitor name-dropping |
| review checklist / weekly review (as noun headline) | Use **weekly reflection** or **look back** |
| goals dashboard / OKR / SMART goals | Productivity-bro framing |
| optimize / hack / crush it | Anti-allyship tone |

## Preferred (game voice)

| Instead of | Use |
|------------|-----|
| Weekly review | Weekly reflection · look back at the week |
| Tasks | Moves · commitments · quests |
| Complete your goals | Tend what you planted · name the next honest step |
| Behind on domains | Some domains are resting · park without loss |
| Inbox | Well · raw capture · what surfaced today |
| Productivity | Throughput · practice · the active loop |
