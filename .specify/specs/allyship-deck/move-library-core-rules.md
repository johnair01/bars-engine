# BARs Engine Core Rules — The Move Library (canonical)

> **Authoritative source.** Provided by the author 2026-06-14. This supersedes inferences in
> `sources-synthesis.md` where they conflict. The move library = **6 Basic Moves × 6 Operations =
> 36 submoves.**

## Fundamental structure

Six **Basic Moves**:

1. **Wake Up**
2. **Open Up**
3. **Clean Up**
4. **Grow Up**
5. **Show Up**
6. **Level Up** *(optional Meta Move)*

Each Basic Move contains **six submoves**, generated by applying the six **Operations** in sequence:
**Shaman → Challenger → Regent → Architect → Diplomat → Sage.**

> **The Faces are not classes. The Faces are operations.** Every move recursively passes through all
> six operations. (This corrects the earlier "faces = classes/lenses" framing.)

## The Six Operations

| Operation | Verb | Question |
|-----------|------|----------|
| **Shaman** | Notice | What is here? |
| **Challenger** | Challenge | What resists? |
| **Regent** | Steward | What deserves responsibility? |
| **Architect** | Amplify | What value wants to be increased? |
| **Diplomat** | Care | What relationships or power dynamics matter? |
| **Sage** | Integrate | What larger truth is emerging? |

## The Basic Moves (purpose · question · output BAR)

| # | Move | Purpose | Question | Output |
|---|------|---------|----------|--------|
| 1 | **Wake Up** | Detect charge | What is happening? | **Awareness BAR** |
| 2 | **Open Up** | Receive charge | What energy is trying to get through? | **Experience BAR** + Channel Identified |
| 3 | **Clean Up** | Transform charge | What move is missing? | **Insight BAR** + Missing Move Identified |
| 4 | **Grow Up** | Develop capability | Who must I become? | **Wisdom BAR** + Capacity Gained |
| 5 | **Show Up** | Invest capacity | What shall I create? | **Artifact BAR** |
| 6 | **Level Up** | Advance developmental position | What is complete? | **Face Advancement** |

Show Up artifacts: Quest · Daemon · Gift · Story · Ritual · **Deck Card** · Contact · Plan · Event.

## The 36 submoves (Move × Operation)

### 1 · Wake Up — *Notice charge → Awareness*
- **Shaman**: Notice the signal. *What is here?*
- **Challenger**: Notice resistance. *What resists being seen?*
- **Regent**: Notice stewardship. *What deserves attention?*
- **Architect**: Notice potential. *What value wants to increase?*
- **Diplomat**: Notice relationships. *What relational dynamics matter?*
- **Sage**: Notice meaning. *What larger pattern is emerging?*

### 2 · Open Up — *Receive charge → Channel Identification*
- **Shaman**: Allow experience. *What am I actually feeling?*
- **Challenger**: Allow discomfort. *What am I avoiding feeling?*
- **Regent**: Hold responsibility. *Can I stay with this?*
- **Architect**: Receive the resource. *What energy is hidden here?*
- **Diplomat**: Care for experience. *How can I relate compassionately to this?*
- **Sage**: Witness experience. *What happens when I stop fighting it?*

### 3 · Clean Up — *Transform charge → Alchemical Transformation*
- **Shaman**: Identify channel. *Which channel is active — Fire / Water / Metal / Earth / Wood?*
- **Challenger**: Challenge interpretation. *What story am I believing?*
- **Regent**: Identify missing move. *What capability is unavailable?*
- **Architect**: Select transformation. *Transcend / Translate / Neutralize?*
- **Diplomat**: Choose destination. *Which channel would better serve this situation?*
- **Sage**: Extract insight. *What does this teach me?*

### 4 · Grow Up — *Develop capability → Capacity*
- **Shaman**: Identify emerging capacity. *What wants to grow?*
- **Challenger**: Identify developmental edge. *What must evolve?*
- **Regent**: Steward growth. *What deserves practice?*
- **Architect**: Amplify capacity. *What capability wants strengthening?*
- **Diplomat**: Relate growth. *How does this affect others?*
- **Sage**: Integrate growth. *Who am I becoming?*

### 5 · Show Up — *Invest capacity → Artifact*
- **Shaman**: Choose domain. *Where will I invest this — Gather Resources / Raise Awareness / Direct Action / Skillful Organizing?*
- **Challenger**: Create intervention. *What must change?*
- **Regent**: Create stewardship. *What deserves ongoing support?*
- **Architect**: Create structure. *What value am I amplifying?*
- **Diplomat**: Create relationship. *Who must be involved? What power dynamic must be addressed?*
- **Sage**: Create legacy. *What artifact remains?*

### 6 · Level Up — *Advance developmental position → Face Advancement*
Faces advance as developmental levels: **Shaman → Challenger → Regent → Architect → Diplomat → Sage**;
completion of one face unlocks the next. Cross-move progression spirals: *Sage·Shaman → Shaman·Challenger;
Sage·Challenger → Shaman·Regent; …* (operation-within-level × level = the 6×6 developmental spiral).

## The Complete BAR Flow (core game loop)

```
Charge → Wake Up (Awareness) → Open Up (Energy) → Clean Up (Move)
       → Grow Up (Capacity) → Show Up (Artifact) → Level Up (Development)
```

## Design Principle (Capability Model — canonical)

> The player is never trying to eliminate emotion. The player is restoring capability.

| Channel | Capability |
|---------|-----------|
| **Fire** | Agency |
| **Water** | Connection |
| **Metal** | Exploration |
| **Earth** | Rest |
| **Wood** | Participation |

The purpose of the game is to **discover and deploy the moves that restore those capabilities.**

## Deck implications (derived)

- **The Allyship Deck's move library IS the 36 submoves** (6 moves × 6 operations). That is the
  card set — a natural ~36-card deck, fully enumerated above, no invention required.
- **Card = one submove**, written in the skill-stack anatomy (Primary Question / Optimizes For /
  Forbidden Moves / Failure Modes / Remediation), seeded from the verb + question above.
- **Operations (faces) are channel-agnostic** — confirmed: channel is *selected inside Clean Up*
  (Shaman Clean Up identifies it; Architect Clean Up picks Transcend/Translate/**Neutralize**),
  not fixed per face.
- **Domains** (Gather Resources / Raise Awareness / Direct Action / Skillful Organizing) are
  *selected inside Show Up* (Shaman Show Up).
- **Consult axes:** by Basic Move ("what stage am I in?"), by Operation ("notice / challenge /
  steward / amplify / care / integrate"), by capability offline ("I can't connect"), or draw at
  random. Level Up is the developmental progression layer.
- Possible "major" cards on top of the 36: 6 Basic Move cards + 6 Operation cards + 5 Capability
  cards (deck stays lean; these are optional framing cards).
- **Correction to earlier notes:** Open Up is **Basic Move #2** (not "the 5th move"); Level Up is
  the optional 6th meta-move. `Neutralize` is a third transformation type alongside Transcend/Translate.
