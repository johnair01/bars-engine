# Admin Thresholds: Bruised Banana Fundraiser

Recommended thresholds for advancing Kotter stage. Admin uses as guidance; advancement remains manual.

| Stage | Name | Donation % | Backlog Items Done | Notes |
|-------|------|------------|--------------------|-------|
| 1 | Urgency | 0% | 0 | Launch |
| 2 | Coalition | ~15% | 3+ | Allies engaged |
| 3 | Vision | ~30% | 6+ | Clear direction |
| 4 | Communicate | ~45% | 9+ | Spread the word |
| 5 | Obstacles | ~60% | 12+ | Address blockers |
| 6 | Wins | ~75% | 15+ | Milestones reached |
| 7 | Build On | ~90% | 18+ | Scale the change |
| 8 | Anchor | 100% | 21+ | Sustainable |

**How to use**: When both metrics reach or exceed the threshold for the next stage, admin can advance the instance's kotterStage in Admin → Instances.

**Customize**: Adjust these values for your specific campaign. Backlog items = Spec Kit backlog items marked Done.
