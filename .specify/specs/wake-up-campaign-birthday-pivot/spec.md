# Wake-Up Campaign (Birthday Pivot)

## Purpose
Define campaign as a narrative revenue funnel and a seamless onboarding bridge. It serves as a landing page to orient new users to the game while developing their specific in-game narrative based on their CYOA choices, maximizing engagement for the allyship campaign.

## Structure
5 Acts:
1. Fracture (Urgency)
2. Spiral (Vision)
3. Greenhouse (Quick Wins)
4. Ask (Action)
5. Oath (Anchoring)

Each Act must define:
- Narrative goal
- Kotter stage mapping
- Hexagram flavor
- CTA integration
- Twine node structure

## Twine Requirements
- Linear spine with branching flavor paths.
- All paths reconverge per Act.
- Each Act produces artifact or reflection node.
- Final Act includes donation + coaching + dev options.

## Integration Hooks
- Optional vibeulon mint on completion.
- Optional ledger entry.
- No required complex game-state coupling.

## Non-Goals
- No new economy rules.
- No new governance mechanics.
- No Sage logic.
- No developmental scoring.
