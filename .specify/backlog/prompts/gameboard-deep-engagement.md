# Prompt: Gameboard Deep Engagement — 3-Step Completion, Steward Visibility, Hexagram Alignment

**Use this prompt when implementing deep-engagement gameboard flow: Wake Up → Clean Up → Show Up, steward visibility, bidding/AID/fork, and hexagram-aligned campaign-throughput quest generation.**

## Context

The gameboard currently makes Complete the star. Players can complete without reading. We need: (1) 3-step completion flow so players must read and reflect before completing, (2) steward visibility so the community sees who holds each quest, (3) vibeulon bidding to unblock, AID offers, and private forks, (4) hexagram-aligned quest generation that answers "How does [stage action] tie to people showing up?"—without Kotter stage names in titles.

## Prompt text

> Implement the Gameboard Deep Engagement spec per [.specify/specs/gameboard-deep-engagement/spec.md](../specs/gameboard-deep-engagement/spec.md). Add 3-step completion (Wake Up, Clean Up, Show Up); steward model with name visibility; vibeulon bidding; offer AID and fork; generateCampaignThroughputQuest using hexagram + campaign goal + stage action. No Kotter stage names in generated quest titles.
