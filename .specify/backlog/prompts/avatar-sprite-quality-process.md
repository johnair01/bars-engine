# Prompt: Avatar Sprite Quality Process

**Use when developing high-quality avatar sprites that match Gathertown/Stardew Valley aesthetic and game vibe.**

## Context

Current sprites feel disconnected. Vision: avatars like Gathertown (retro pixel, customizable, friendly) and Stardew Valley (cozy, layered, expressive), while fitting the game's playful, ironic, inviting tone and Construct Conclave's elemental nations/archetypes.

## Prompt text

> Implement the avatar sprite quality process per [.specify/specs/avatar-sprite-quality-process/spec.md](../specs/avatar-sprite-quality-process/spec.md). Create the style guide first (aesthetic target, palette, dimensions, registration). Then develop base layer, nation layer, playbook layer in order. Ensure modularity: all layers align to same anchor, transparency allows stacking, nation/playbook accents reflect handbook lore. Test nation × playbook combos. Update docs/SPRITE_ASSETS.md with style guide and quality checklist.

## Checklist

- [ ] STYLE_GUIDE.md: Gathertown/Stardew feel + game tone
- [ ] Color palette (8–16 colors)
- [ ] Dimension decision (64×64 vs 32×32)
- [ ] Base layer: default, male, female, neutral
- [ ] Nation layer: 5 nations × body + accent
- [ ] Playbook layer: 8 playbooks × outfit + accent
- [ ] Combo testing
- [ ] docs/SPRITE_ASSETS.md update
