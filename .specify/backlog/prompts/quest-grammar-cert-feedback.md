Certification feedback (cert-quest-grammar-v1 STEP_1):

> as for the 6 unpacking questions. we need a mix of dropdown choices and space for players to type short responses. Even though this is an admin tool I want it to follow the design philosphy of mobile first, and using the choose your own adventure model along with the emotional tone and aspect.

> ok I compiled and previewed and the nodes that have been created are a bit of nonsense. we need to analyze how these nodes are being created and switch things up to make sure that what is being generated is gramatical. Once the structure looks good then an admin should be able to edit the output and then there is a final layer where they generate based on teh deterministic structure a quest that makes sense and produces the right kind of BARs and quests.
