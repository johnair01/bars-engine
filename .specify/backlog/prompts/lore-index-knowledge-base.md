# Prompt: Lore Index and Knowledge Base

**Use this prompt when implementing the lore index and knowledge base for onboarding and invitations. Unblocks Event-Driven CYOA (AC) by providing canonical lore reference.**

## Prompt text

> Implement the Lore Index and Knowledge Base per [.specify/specs/lore-index-knowledge-base/spec.md](../specs/lore-index-knowledge-base/spec.md). Create a canonical proper-noun index (content/lore-index.md), a wiki index page at /wiki, and knowledge base pages for nations, archetypes, allyship domains, 4 moves, Bruised Banana campaign, and glossary. Add a "Learn more" link from the Event page Wake Up section to the wiki. Use game language: WHO (nations, archetypes), WHAT (quests, BARs), WHERE (allyship domains), Energy (vibeulons), personal throughput (4 moves).
