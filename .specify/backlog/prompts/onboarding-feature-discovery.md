## Summary

Clean up the onboarding send experience so players know how to make BARs, create quests, use EFA, and donate. Content must exist as **quests** (actionable) and **documentation** (reference). Goal: excite players to use the app and move the needle.

## Key Deliverables

- **BARs**: Quest + docs—what a BAR is, how to make one, how it connects to gameplay
- **Quests**: Quest + docs—how to make a quest, add subquests to campaign quests
- **EFA**: Quest + docs—how to use Emotional First Aid
- **Donation**: Quest/prompt + docs—how to donate to the campaign; visible link
- **Send flow**: Game Map → first BAR → first quest → EFA → donation; inviting tone

## Dependencies

- Game Map and Lobby Navigation (FP)
- Dashboard Orientation Flow
- K-Space Librarian (if applicable)
- Event/Donation flow
