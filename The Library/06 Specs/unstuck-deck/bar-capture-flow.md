# BAR Capture Flow

## Purpose

Preserve the clearest insight from a card practice as reusable pattern language.

## Capture Sequence

1. Name the pattern in one sentence.
2. Name the protection underneath it.
3. Name the cost of keeping it unchanged.
4. Name the smallest honest next move.
5. Save the phrase as a reusable BAR seed.

## Draft BAR Seed Shape

```json
{
  "pattern_phrase": "I use design to postpone exposure to reality.",
  "protective_intelligence": "I am trying to stay safe from a public miss.",
  "cost": "The work never gets real feedback.",
  "next_move": "Run one 24-hour test."
}
```
