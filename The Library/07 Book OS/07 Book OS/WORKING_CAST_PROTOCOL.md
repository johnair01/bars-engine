# Working Cast Protocol

## Purpose

This note keeps the lightweight casting skill used in Book OS design and editorial sessions.

Use this when a full house ritual is not available but a cast is still needed to set the lens for a 6-face or ITD process.

## Method

Working method:

- 3-coin cast
- 6 lines
- bottom to top
- changing lines enabled

Coin values:

- Heads = `3`
- Tails = `2`

Line values:

- `6` = old yin, changing
- `7` = young yang
- `8` = young yin
- `9` = old yang, changing

## Use Rule

The cast is not decoration.

The cast is the lens all 6 faces use to justify their positions.

Sequence:

1. Cast
2. Name primary hexagram
3. Name changing lines
4. Name relating hexagram
5. Use the cast to frame the 6-face consultation
6. If doing ITD, cast before running the audit
