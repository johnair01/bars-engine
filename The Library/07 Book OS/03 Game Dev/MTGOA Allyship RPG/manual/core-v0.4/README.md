# Core v0.4 - School Ship Engine

This folder is the current canonical design entry point for the MTGOA RPG.

## Current Core Claim

**Mastering the Game of Allyship is a school-ship RPG where students of allyship travel from world to world, resolving crises through four enduring practice Houses while advancing through six Schools of mastery.**

## Decisions Locked For Now

- The four allyship domains are Houses / practice factions, not districts.
- The school-ship is the recurring home base.
- Campaign packets provide the worlds the ship visits.
- The Headmaster is a recurring NPC with philosophy, grief, blind spot, and mission authority.
- The six Faces become Schools / mastery milestones.
- The six Schools are horizontal paths with six altitude ranks inside each, not a simple linear ladder.
