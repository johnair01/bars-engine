# Wendell Design Pass - School Ship Fork

Status: raw design pass captured for integration

Date: 2026-06-11

## Core Setting Direction

The setting should be a school.

The four allyship domains are more like Hogwarts Houses / factions than physical districts.

There is a Headmaster NPC.

Vibe mix:

- Avatar: The Last Airbender
- Harry Potter
- X-Men
- post-Order-66 Jedi Temple in hiding

The plot takes place on a flying ship moving from planet to planet helping people.

Image:

After Order 66, imagine the Jedi Temple could leave orbit and go into hiding.

The ship gets recreated from a vibes standpoint each time, but the four domains remain across all *Mastering Allyship* campaigns.

## World Creation

The v0.2 "Creating the City" section does something useful: it asks questions that make the world emerge.

But the manual needs to actually walk players through the experience of answering those questions so that, by reading the book, they can see how the world emerges from questions.

## Character Creation

"Calling" is unclear.

It might connect to the superpower archetypes.

Ensemble Role can use the five-man-team structure from Hearts Blazing.

Private Motive is probably closer to a subclass feature connected with the self-sabotaging beliefs.

Cost and Line are not obviously universal, but after reading further they do seem good for character building.

If Cost / Line / Bond stay, each should probably unlock or define a special move.

Shared Stake is still unresolved. Allyship campaigns should probably cover that.

## Faces and School Structure

Faces of the Game Master reflect milestones in one's allyship journey.

They can be seen as Pokemon Gym Leaders of each level of the school.

This implies six schools.

Face Gifts need more mechanics.

Possible direction:

- player creates their own
- group brainstorm stage decides the outcome
- tied to school / milestone / teacher structure

## Self-Sabotaging Beliefs

The six self-sabotaging beliefs:

- I'm not Good Enough
- I'm not ready
- I don't belong
- I'm insignificant
- I'm not worthy
- I'm not capable

Private Motive may connect to these as subclass/shadow features.

## Stats and Moves

Stats still need to be developed.

If stats connect to the basic moves, this needs expansion.

The basic moves are categories of moves, not the whole move list.

Players should inherit moves from:

- class / role
- subclass / belief pattern
- level / Face milestone

These inherited moves slot into Wake Up, Clean Up, Grow Up, and Show Up.

## Pools

Capacity, Trust, Leverage, and Clarity are interesting but not yet clear.

Need to define:

- what they mean
- how players spend them
- how they refresh
- why they are fun
- whether all four are necessary

## Benefits

Connecting benefits to basic moves is a good idea.

Instead of one shared benefit list, each move category should have its own benefits:

- Wake Up benefits
- Clean Up benefits
- Grow Up benefits
- Show Up benefits

This tightens narrative and mechanical outcomes.

## Adversity Tokens

Need to think more.

Dungeon World marks XP on misses.

Are Adversity Tokens doing XP work?

The GM should be offering these tokens.

If there is no GM, players have to give tokens to each other for balance.

## Tension Tokens

The system-administered nature of Tension Tokens is good.

They fulfill the requirement that tokens are emergent consequences of play.

## Support Moves

Support Moves are interesting.

Question:

Does each class and subclass have its own Support Move?

## Episode Threshold

Episode Threshold and Role Cards may be running into the problem of also having a GM resolve play.

Need to clarify whether bidding is:

- a guide-less narration tool
- a pre-scene pressure mechanic
- a player spotlight mechanic
- or unnecessary when a Guide exists

## Static

Static is not landing.

Current version feels nonsensical as a mechanic and needs revisiting.

Likely replacement:

- faction clocks
- ship condition
- mission pressure
- corruption / drift / exposure
- threat clocks
- relationship heat

## Episodic vs Downtime Play

The system needs a clear difference between:

- episodic play
- downtime play between episodes

Downtime may be where school, training, relationships, advancement, and campaign prep live.
