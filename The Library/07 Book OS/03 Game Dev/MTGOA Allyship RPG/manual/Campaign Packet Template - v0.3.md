# Campaign Packet Template - v0.3

Use this to create sellable campaign content for the MTGOA RPG engine.

## 2. Premise

Write one paragraph.

Include:

- where the campaign takes place
- who the player characters are
- what pressure starts the story
- why allyship matters here

## 3. The Field

The Field is the playable social world.

Include:

- map or relationship diagram
- 6 to 10 locations
- 3 visible institutions
- 3 hidden pressures
- 1 central contradiction
- 1 public symbol
- 1 private cost

## 4. Practice Factions

Rename these in setting-native language.

| Engine Slot | Setting Name | Function |
|---|---|---|
| Provisioners |  | Gather Resources |
| Weavers |  | Skillful Organizing |
| Linekeepers |  | Direct Action |
| Lanternbearers |  | Raise Awareness |

For each faction:

- public face
- real need
- current project
- resource
- boundary
- shadow move
- relationship to the other three
- clock

## 5. Local Factions

Create 3 to 6 factions that belong only to this campaign.

For each:

- name
- desire
- fear
- leverage
- what they will not say directly
- who they depend on
- who they resent
- clock

## 6. NPC Roster

Create 12 named NPCs.

For each:

- name
- role
- desire
- fear
- need
- boundary
- leverage
- first impression
- what changes if treated with dignity

## 7. Fronts

Create 3 campaign fronts.

For each:

- name
- impulse
- cast
- stakes question
- clock
- moves
- grim future

Create 2 session fronts for the Pilot Episode.

## 8. Episode Deck

Create:

- 1 Pilot Episode
- 8 to 12 Midseason Episodes
- 2 Finale Episodes
- 4 Static cards

Each Episode has:

- title
- in-focus Role
- opening pressure
- involved factions
- supporting cast
- keywords
- threshold
- crisis clock
- question

## 9. Custom Moves

Create 3 to 6 setting-specific moves.

Move template:

When you ___, roll + ___.

On a 10+, ___.

On a 7-9, ___, but ___.

On a 6-, ___.

## 10. Fast Start

Write the one-shot version.

Include:

- 20-minute setup
- opening scene
- 3 required beats
- 1 optional complication
- finale trigger
- debrief questions
- real-world rep

## 11. Deep Start

Write the campaign setup.

Include:

- tone questions
- faction customization
- map prompts
- relationship questions
- starting clocks
- first Pilot Episode

## 12. Practice Bridge

For each session, include:

- BAR prompt
- debrief question
- real-world rep
- optional journaling prompt

## 13. Product Notes

Define:

- expected play time
- player count
- content warnings
- replayability hooks
- what makes this campaign distinct
- what engine mechanics it showcases
