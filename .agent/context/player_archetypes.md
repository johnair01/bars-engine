# Player Archetypes & The Vibeulon Engine

> *"The first rule of vibeulons is that the vibes must flow."*

---

## The Deep Structure

```
HEXAGRAM = UPPER TRIGRAM + LOWER TRIGRAM
        = WHERE YOU'RE GOING + WHERE YOU ARE
        = TARGET STAGE + CURRENT STAGE
```

Each I Ching reading produces a hexagram—two archetypes in relationship. This teaches players to embody their most effective self at the right time, in alignment with the sage.

**This is the essence of Teal consciousness**: seeing the whole system, knowing when to act from which center, trusting the emergent wisdom.

---

## 4 Moves vs 8 Archetype Moves

**4 moves (personal throughput)** — how players get things done: Wake Up (see more), Clean Up (emotional energy/unblocking), Grow Up (skill capacity), Show Up (completing quests). See [terminology.md](terminology.md).

**8 archetype moves** (below) — Kotter stages, mapped to Enneagram × Trigram. Distinct from the 4 moves.

## Hexagram Quest Mechanics

When a quest is cast, the hexagram reveals:

```
┌─────────────────┐
│  UPPER TRIGRAM  │  ← Target State (Where to go)
│    (Stage N+1)  │
├─────────────────┤
│  LOWER TRIGRAM  │  ← Current State (Where you are)
│    (Stage N)    │
└─────────────────┘
```

### Example: Hexagram 11 (Peace/Tai)

```
☷ Earth (above)  → Stage 2: Coalition
☰ Heaven (below) → Stage 3: Vision
```

**Reading**: You have the vision (Heaven/Reformer energy below). Now build the coalition (Earth/Helper energy above). The sage says: share your clarity with others who can help manifest it.

**Player Prompt**: "Which Helper in your circle can carry this vision forward?"

| Level | Learning |
