# Quest Template Types

TypeScript types for the template extraction engine.

## Intended Types

- `QuestTemplate` — Full template model
- `TemplateSummary` — List view
- `NodePattern`, `ActionPattern`, `BarPattern`, `CompletionPattern` — Structural abstractions
- `ExtractParams`, `ExtractResult` — Extraction request/response
- `ExtractionReport` — Report payload

See [quest-template-api.md](../../../docs/architecture/quest-template-api.md) for payload shapes.
