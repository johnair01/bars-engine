# Archetype Agent World Loop Examples

## Example 1 — Charge to Quest

| Step | Actor | Action |
|------|-------|--------|
| 1 | Player | Creates high-charge BAR ("Frustration about housing costs") |
| 2 | Danger Walker | Proposes exploration quest (wake-up) |
| 3 | Truth Seer | Synthesizes issue context from charge |
| 4 | Joyful Connector | Suggests one collaborator for the quest |

**Result**: Charge flows into structured quest pathway with collaboration invitation.

---

## Example 2 — Campaign Stall

| Step | Actor | Action |
|------|-------|--------|
| 1 | System | Campaign shows no new activity for 5 days |
| 2 | Devoted Guardian | Proposes support check-in |
| 3 | Decisive Storm | Suggests activation event |
| 4 | Still Point | Suggests whether clean-up is needed before acceleration |

**Result**: Stalled campaign receives layered interventions (support, activation, grounding).

---

## Example 3 — Single Player Ecology

| Step | Actor | Action |
|------|-------|--------|
| 1 | Player | Records a BAR (solo, no other humans) |
| 2 | Agent set | Produces suggestions, appreciation, quest pathways |
| 3 | World | Feels inhabited even without many human users |

**Agents involved**:
- Truth Seer: reflection BAR or pattern summary
- Joyful Connector: appreciation for completing a quest
- Danger Walker: exploration quest from charge
- Devoted Guardian: support if player seems stuck

**Result**: Solo player experiences world aliveness through archetypal responses.

## Example 4 — Event Production Support

| Step | Actor | Action |
|------|-------|--------|
| 1 | Event Campaign | Reaches "Invite collaborators" milestone |
| 2 | Joyful Connector | Suggests invite waves |
| 3 | Decisive Storm | Proposes promotional push before event deadline |
| 4 | Post-event | Truth Seer creates summary BAR; Danger Walker identifies unresolved tensions |

**Result**: Event production receives archetypally coherent support across lifecycle.

---

## Example 5 — Quest Completion Celebration

| Step | Actor | Action |
|------|-------|--------|
| 1 | Player | Completes quest (public or campaign-visible) |
| 2 | Joyful Connector | Creates appreciation BAR |
| 3 | Truth Seer | Optionally synthesizes learning from completion |

**Result**: Completion is acknowledged; world feels responsive.

---

## Loop Invariants

- Agents do not create feedback spam: throttling limits apply
- Multiple agents can act without overlapping excessively: rule priority and archetype specialization
- Single-player scenarios remain coherent: agent set produces bounded, legible responses
- Private data is never leaked: agents only act on visibility-permitted content
