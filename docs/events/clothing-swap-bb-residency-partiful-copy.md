## Full description

We’re running a **clothing swap** as a **fundraiser** for the Bruised Banana residency.

**How it works**

- **Bring / post clothes** — photos + a short story (a BAR) about the piece; optional details (brand, size, condition) in-app.
- **One shared gallery** — everyone browses the same pool (IRL and remote).
- **Bidding** — **vibeulons** (in-app energy) with a **single event closing time**; you can also make **BAR offers** (bigger trades or services). **Sellers choose** whether to accept a BAR offer; if they don’t accept before close, the **highest vibeulon bid** wins that item.
- **Donate** — we’ll prompt at **kickoff** and **close**; you can support even if you can’t attend.

**Hybrid**

- **In person:** bring items, try things on, meet the field.
- **Online:** follow the same gallery and participate in bidding / offers from wherever you are.
